프로젝트 설명
===
첫 회사에서 파견 나가 접하게 된 프로젝트가 대형 쇼핑몰 프로젝트였습니다.   
참여한 프로젝트 내에서 모든 기능을 접해보지 못하였지만, 생소한 프레임워크나 라이브러리 등을 개인 프로젝트를 통해 공부해보고자   
<code>Front-end</code>와 <code>Back-end</code> 부분을 모두 담당하며 쇼핑몰이란 주제의 프로젝트를 진행하였습니다.
